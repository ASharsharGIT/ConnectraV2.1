## ðŸ› Troubleshooting

### Modal Not Opening?
1. Check ID matches: `data-modal-trigger="myModal"` â†’ `id="myModal"`
2. Verify modal structure has `.modal-overlay` and `.modal-container`
3. Check console for JavaScript errors

### Card Not Expanding?
1. Add `.expandable-card` class
2. Include `.card-preview` and `.card-expanded` sections
3. Add `.expand-trigger` button

### Accordion Not Working?
1. Wrap in `.accordion-container`
2. Items need `.accordion-item` class
3. Headers need `.accordion-header` class
4. Content needs `.accordion-content` class
